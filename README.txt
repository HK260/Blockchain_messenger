Blockchain Messenger
====================

Introduction
------------

The Blockchain Messenger is a smart contract designed to leverage the Ethereum blockchain for secure and transparent message storage. This contract enables messages to be stored permanently on the blockchain, viewable by all initially but with subsequent interactions restricted to the contract's creator.

Features
--------

*   **Blockchain-based Messaging:** Utilizes Ethereum blockchain technology to store messages securely and permanently.
*   **Privacy and Access Control:** While the initial message is public, further interactions are restricted to the contract creator, ensuring privacy and control.
*   **Update Count:** The contract automatically tracks and displays the number of times the message has been updated, enhancing transparency.
